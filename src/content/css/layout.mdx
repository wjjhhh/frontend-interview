## 核心概念

### BFC（块级格式化上下文）

BFC 是一个独立的渲染区域，内部元素的布局不影响外部。

**触发条件：**
- `overflow` 不为 `visible`
- `display: flow-root`
- `float` 不为 `none`
- `position: absolute/fixed`
- `display: inline-block/flex/grid`

**应用场景：**
- 清除浮动
- 防止 margin 重叠
- 自适应两栏布局

## 高频面试题

### 1. 水平垂直居中的方案

```css
/* 1. Flex 方案（推荐） */
.parent {
  display: flex;
  justify-content: center;
  align-items: center;
}

/* 2. Grid 方案 */
.parent {
  display: grid;
  place-items: center;
}

/* 3. 绝对定位 + transform */
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

/* 4. 绝对定位 + margin auto */
.child {
  position: absolute;
  inset: 0;
  margin: auto;
  width: fit-content;
  height: fit-content;
}
```

### 2. Flex 布局核心属性

```css
/* 容器属性 */
.container {
  display: flex;
  flex-direction: row;         /* 主轴方向 */
  flex-wrap: wrap;             /* 换行 */
  justify-content: space-between; /* 主轴对齐 */
  align-items: center;         /* 交叉轴对齐 */
  gap: 16px;                   /* 间距 */
}

/* 项目属性 */
.item {
  flex: 1 0 200px; /* flex-grow flex-shrink flex-basis */
  align-self: flex-start;
  order: 1;
}
```

### 3. Grid 布局实战

```css
/* 经典的圣杯布局 */
.layout {
  display: grid;
  grid-template-areas:
    "header header header"
    "nav    main   aside"
    "footer footer footer";
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}

.header { grid-area: header; }
.nav    { grid-area: nav; }
.main   { grid-area: main; }
.aside  { grid-area: aside; }
.footer { grid-area: footer; }
```

## 深入追问

### CSS 容器查询 vs 媒体查询

```css
/* 媒体查询 - 基于视口 */
@media (max-width: 768px) { ... }

/* 容器查询 - 基于父容器（CSS Container Queries） */
.card-container {
  container-type: inline-size;
}

@container (max-width: 400px) {
  .card { flex-direction: column; }
}
```

### 新的 CSS 特性

- `subgrid`：继承父 Grid 的轨道定义
- `:has()` 选择器：父选择器
- `@layer`：控制样式优先级
- `color-mix()`：颜色混合函数
